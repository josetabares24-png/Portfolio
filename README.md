# Portfolio

One-page portfolio/landing site built with React 19 and Vite.

## Stack

- React 19 + Vite
- Tailwind CSS v4
- shadcn/ui, with components from [Cult UI](https://cult-ui.com)
- Framer Motion for section transitions

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
